# 🗂️ Smart Downloads Organizer

**CLI-утилита для автоматической сортировки и очистки папки «Загрузки».**

Этот проект решает проблему хаоса в папке загрузок. Скрипт анализирует файлы, раскладывает их по категориям (Изображения, Документы, Инсталляторы) и архивирует старые данные. Написан с упором на безопасность (режим сухой проверки) и расширяемость (YAML-конфигурация).

## ✨ Возможности

- **📁 Автоматическая сортировка:** Распределение файлов по папкам на основе расширений.
- **⚙️ Гибкая конфигурация:** Правила сортировки хранятся в `config.yaml`, легко редактируются без изменения кода.
- **🛡️ Безопасный режим (Dry-Run):** Возможность просмотреть план действий перед реальным перемещением файлов.
- **📦 Архивация:** Автоматическое сжатие файлов, которые не открывались более N дней.
- **📝 Логирование:** Все действия записываются в лог-файл для аудита.
- **💻 Кроссплатформенность:** Работает на Windows, macOS, Linux.
- **🚀 Готовый .exe:** Возможность сборки в исполняемый файл для запуска без установленного Python.

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
git clone https://github.com/yourusername/smart-organizer.git
cd smart-organizer
pip install -r requirements.txt
```
### 2. Запуск
По умолчанию скрипт использует файл config.yaml в текущей директории.
### Проверка того, что сделает скрипт (без реальных изменений)
```
python main.py --dry-run
```

### Реальная сортировка файлов
```
python main.py
```

### Сортировка + Архивация старых файлов (старше 30 дней)
```
python main.py --archive
```

### 2. Сборка в .exe (для Windows)
Если вы хотите использовать утилиту без установки Python:
```angular2html
pip install pyinstaller
pyinstaller --onefile --name SmartOrganizer main.py
```

## Технический стек
- 🖥️ **Python: 3.8+**
- 🖥️ **CLI Interface:** [Click](https://click.palletsprojects.com/) — для создания удобных командных интерфейсов.
- ⚙️ **Config:** [PyYAML](https://pyyaml.org/) — для парсинга конфигурации.
- 📂 **File System:** `pathlib`, `shutil` — стандартные библиотеки для работы с файлами.
- 📝 **Logging:** `logging` — встроенный модуль для отслеживания событий.
- 📦 **Packaging:** `PyInstaller` — для компиляции в бинарный файл.
